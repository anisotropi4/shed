Open Data for public-transport visualisation

A set of open data and other resources that could be used to produce transport and related visualisations.

Base Geography

The base geography and population data is based on:

  • The base map of mainland Britain is derived from the WorldPop base maps under CC 4.0 by deed retrieved 2023-09-07.
  • OpenStreetMap data is then used to validate and also to identify 10 stations locations which is licensed under CC-BY-SA 2.0 through the OverPassAPI Turbo service.

Base Rail data

These base data consist of the centre-line or network track-model, Network Rail location codes, plus the CIF format timetable and VSTP updates.

Product Note Owner Licence
Track Model Railway network model Shapefile with link, node, reference lines and waymark layers. See Open Data Technical Specification. Network Rail Open (OGL3)
CORPUS Lookup codes for rail stations from the CORPUS system (Codes for Operations, Retail & Planning – a Unified Solution). See OpenRailData wiki Network Rail Open (OGL3)
Schedule Daily train schedule extract and updates from the national Integrated Train Planning System in Common InterFace (CIF) format. See OpenRailData wiki Network Rail Open (OGL3)
Very Short-Term Planning (VSTP) Real-time train schedules created via the Very Short Term Plan (VSTP) process. See OpenRailData wiki Network Rail Open (OGL3)

Key Station and Public Transport Data

These data are the National Passenger Transport Access Node (NaPTAN) and Transport Gazette for Great Britain.

Product Note Owner Licence
NaPTAN & NPTG This CSV and XML API contains information for all passenger transport nodes including rail, bus-stops, ferry terminals and airports. PDF user guide Department for Transport (DfT) Open (OGL3)
Station attributes for all mainline stations in GB Table 6329: Station attributes for all mainline stations, Great Britain as ODS file. Office of Road and Rail (ORR) Open (OGL3)
Stations Reference Data Geographic station names output from journey planning in XML format file. Rail Development Group (RDG) Open
Knowledgebase Stations data feed A station data-feed which includes information on train stations in XML format API. Rail Development Group (RDG) Open
